## Get Element
```js
$('***')
```

## Get Text In Element
```js
$('***').text();
```

## Get Value In Input
```js
$('***').val()
```

## Get Data Attribute Value
```html
<li data-sample="sample">Sample</li>
```
```js
$('***').data('sample');
```

## Get Value Of Location Object Properties
```js
$(location).attr('href');
$(location).attr('search');
```
