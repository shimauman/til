## Image

```
自身のPC --------------------

port:8000 自身のローカルホスト
||
port:80 nginxコンテナ(起動中)
↓↑
port:9000 appコンテナ(起動中)

----------------------------
```

## Note
- Dockerコンテナは起動するだけでよい。
- 自身のローカルホスト(複数コンテナが動くサーバーの役割を担っている)の8000番に、自身のPCからアクセスすれば、そのアプリケーションは動いている。- - appコンテナにログインする必要はない。
