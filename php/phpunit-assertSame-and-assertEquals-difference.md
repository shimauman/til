## `assertSame`
Same with `===`. We should use this.
```php
// Check type and value.
assertSame(mixed $expected, mixed $actual[, string $message = '']);
```

## `assertEquals`
Same with `==`.
```php
// Check only value.
assertEquals(mixed $expected, mixed $actual[, string $message = ''])
```
