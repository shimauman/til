## Use Same Extensions
1. Make `.vscode/extensions.json`.
2. Add recommended extensions in the file.
3. We can also add unrecommended extensions.

```json
{
  "recommendations": [
    "***"
  ],
  "unwantedRecommendations": [
    "***"
  ]
}
```

## Use Same Settings
1. Make `.vscode/extensions.json`.
2. Add settings in the file.
