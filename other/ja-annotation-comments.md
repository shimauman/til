## Comments
- `TODO` : add or modify later
- `FIXME` : need to fix
- `OPTIMIZE` : many unnecessary things.
- `NOTE` : the reason why

```json
        "tag": "TODO",
        "color": "#FF8C00",
        "strikethrough": false,
        "underline": false,
        "backgroundColor": "transparent",
        "bold": false,
        "italic": false
    },
]
```
