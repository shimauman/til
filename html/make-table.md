```html
<table>
    <tr>
        <th>col1</th>
        <th>col2</th>
        <th>col3</th>
    </tr>
    <tr>
        <td>val1</td>
        <td>val2</td>
        <td>val3</td>
    </tr>
    <tr>
        <td>val4</td>
        <td>val5</td>
        <td>val6</td>
    </tr>
</table>
```

- `tr` : table row
- `th` : table header
- `td` : table data
